Ajuda

-Você deve fornecer todos os dados sinalizados com asterisco(*).

Observe que no campo Contato você deve indicar o nome da pessoa que é seu contato no Tribunal de Contas.

Se você é funcionário do Tribunal de Contas basta colocar "TCE".

- Após preencher os dados clique em Salvar.

- Após o salvamento você receberá um e-mail com mais informações para prosseguir seu cadastro.

- Acesse o menu Minhas Apostas

- A tela Minhas Apostas contem uma tabela com todos os jogos e cada linha contém os campos para preencher o placar da aposta referente aquele jogo.

Aqui o jogo Bélgica x Argélia foi preenchido, mas o jogo Brasil x México ainda não.

- Observe que para sua aposta estar finalizada, todos os jogos da tabela deverão está preenchidos. E para verificar isso veja se o informativo, que fica no canto superior direito da tela, indica:"Existem 0 apostas incompletas!"

- Existem 3 formas de consulta: por Participante, por Jogo e por Grupo.

- Você pode ver a aposta, jogos e grupos, do participante selecionado

- Você pode ver a aposta de todos os participantes para o jogo selecionado

- Você pode ver a aposta de todos os participante para o grupo selecionado

- Acesse o menu Simulador

- Na tela Simulador você poderá simular o resultado dos jogos que ainda não foram disputados.

Para simular um resultado basta preencher os campos referente ao placar jogo que se quer simular na tabela Simulador

Se o jogo não possui uma caixa de texto

é porque esse jogo já ocorreu. Nesse caso

o jogo Argentina x Bósnia já ocorreu não

podendo mais simular seu resultado

Se você não souber o nome da seleção basta pôr a seta do mouse sobre a bandeira.

- Mande um e-mail com sua dúvida para email@gmail.com com o assunto "Dúvida"
